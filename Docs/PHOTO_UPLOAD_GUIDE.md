# Photo Upload Guide

## Overview
The photo upload feature allows users to upload profile pictures using both camera capture and gallery selection, with automatic optimization for all devices.

## Features

### Device Support
- **Desktop:** File upload from computer
- **Mobile Camera:** Direct capture
- **Mobile Gallery:** Select from photos
- **Tablet:** Both options available

### Image Processing
- **Automatic Compression:** Reduces file size
- **Optimal Sizing:** Max 800x800 pixels
- **Format Conversion:** JPEG optimization
- **Quality Balance:** 85% compression

### Upload Options
- **Camera Icon:** Open camera (mobile)
- **Browse Button:** Select file (all devices)
- **Drag & Drop:** Desktop supported
- **Paste Image:** From clipboard

## Uploading Photos

### Method 1: Camera (Mobile)
1. Click camera icon
2. Grant camera permission
3. Take photo
4. Confirm or retake
5. Auto-uploads

### Method 2: Gallery/File
1. Click "Choose Photo"
2. Select image file
3. Preview appears
4. Click "Upload"

### Method 3: Drag & Drop (Desktop)
1. Drag image to upload area
2. Drop file
3. Preview shows
4. Click "Upload"

## Photo Requirements
- **Formats:** JPG, PNG, GIF, WebP
- **Max Size:** 10MB
- **Auto-optimized:** No manual resizing needed
- **Aspect Ratio:** Maintained

## Editing Options

### Crop Photo
1. Click "Crop" after upload
2. Adjust crop area
3. Confirm crop
4. Save changes

### Delete Photo
1. Click "Delete Photo"
2. Confirm deletion
3. Default avatar shown
4. Cannot be undone

## Troubleshooting

### Camera Not Working
- Check browser permissions
- Update browser
- Try gallery option
- Restart device

### Upload Fails
- Check file size (max 10MB)
- Verify internet connection
- Try different format
- Clear browser cache

### Photo Blurry
- System auto-optimizes
- Original quality preserved
- View full size on profile
- Contact admin if needed

### Mobile Issues
- Use latest browser
- Allow storage access
- Try both options
- Check app permissions

## Browser Compatibility

- Photos scanned for malware
- Personal data protected
- Secure transmission

## Best Practices
1. Use clear, recent photos
2. Plain background recommended
3. Face clearly visible
4. Professional attire preferred
5. Update photo annually

- Steady hand for camera shots
