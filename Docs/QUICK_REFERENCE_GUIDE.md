## Common Issues & Solutions

### Login Problems
- **Forgot PIN:** Click "Forgot PIN" link
- **Account Locked:** Contact administrator
- **Wrong Branch:** Check user assignment
- **Browser Issues:** Clear cache, try another

### Data Entry
- **Validation Errors:** Check required fields
- **Duplicate Entry:** Search existing records
- **Permission Denied:** Verify user role
- **Save Failed:** Check internet connection

### Reports
- **No Data:** Check date filters
- **Export Error:** Try different format
- **Slow Loading:** Reduce date range
- **Print Issues:** Use PDF export
