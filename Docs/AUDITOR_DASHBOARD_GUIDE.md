# Auditor Dashboard Guide

## Overview
The Auditor Dashboard provides comprehensive financial oversight and audit capabilities for monitoring church finances across all branches.

## Access
- **URL:** `/auditing/dashboard/`
- **Permissions:** Auditor, Mission Admin
- **Menu:** Auditing â†’ Dashboard

## Features

### 1. Financial Overview
- **Total Contributions:** Sum of all contributions
- **Mission Allocations:** Amount allocated to mission
- **Total Expenditures:** All expenses across branches
- **Net Balance:** Overall financial position

### 2. Quick Actions
- **Financial Audit:** Detailed financial reports
- **Contributions Trail:** Transaction history
- **Expenditures Trail:** Expense tracking
- **Variance Analysis:** Performance vs targets

### 3. Hierarchical Filtering
- Filter by Area, District, or Branch
- Dynamic dropdown selection
- Real-time data updates

### 4. Recent Activity
- **Recent Contributions:** Latest 10 contributions
- **Recent Expenditures:** Latest 10 expenses
- Click to view full details

### 5. Branch Performance
- **Branch Summary:** Financial data per branch
- **Remittance Status:** Track mission payments
- **Net Balance:** Branch financial health

## Reports

### Financial Audit Report
- Date range selection
- Branch-by-branch breakdown
- Contribution/expenditure analysis
- Excel export available

### Contribution Audit Trail
- Complete transaction history
- Member-wise contributions
- Edit status indicators
- Filter and search options

### Expenditure Audit Trail
- All expense transactions
- Category-wise breakdown
- Approval status tracking
- Export capabilities

### Variance Analysis
- Target vs actual performance
- Monthly/quarterly/yearly views
- Top performers identification
- Branches needing attention

## Export Features
- **Excel Export:** Detailed data with formatting
- **PDF Reports:** Print-ready summaries
- **Custom Filters:** Export specific data ranges

## Audit Alerts
- Missing remittances
- Unusual variances
- Pending approvals
- System notifications

## Best Practices
1. Regular review of branch performance
2. Monitor remittance compliance
3. Investigate large variances
4. Export reports for documentation
5. Maintain audit trail

## Keyboard Shortcuts
- **Ctrl+E:** Export current report
- **Ctrl+F:** Open filter panel
- **Ctrl+R:** Refresh data

## Support
For technical support or audit queries, contact the system administrator.
