

# ## **18. Member Role – Access, Permissions & Functionalities**

Members are the **core users** of the SDSCC system.
They represent the **final level of the church hierarchy**, with access strictly limited to **their own data**, **general church content**, and **allowed financial views**.

Members **cannot** access or view anything outside their personal scope.

---

# ### **18.1 Member Access Level Summary**

| Module                              | Member Permission          |
| ----------------------------------- | -------------------------- |
| Own Contributions                   | Full view (no edit)        |
| Contribution Types List             | View only (if allowed)     |
| General Offering Summaries          | View only                  |
| Funeral Contributions               | View own only              |
| Branch Announcements                | Full view                  |
| District/Area/Mission Announcements | View only                  |
| Sermons                             | Full access                |
| Attendance                          | View own attendance        |
| Member Profile                      | View + edit limited fields |
| Groups/Departments                  | View assigned groups       |
| Pastoral Notes                      | Not visible                |
| Branch Finances                     | No access                  |
| Branch Expenditures                 | No access                  |
| Payroll                             | No access                  |
| Remittance                          | No access                  |
| Utilities & Welfare                 | No access                  |
| Audit Logs                          | No access                  |
| System Settings                     | No access                  |

---

# ### **18.2 Member Dashboard**

The Member Dashboard is designed to provide **simple, personal insights** in a clean, mobile-first layout.

### **Dashboard Sections**

#### **1. Personal Contribution Overview**

* Total individual contributions (year-to-date)
* Breakdown by type:

  * Tithe
  * Pledges
  * Thanksgiving
  * Special projects
* Monthly contribution graph
* Previous year comparison

#### **2. Attendance Overview**

* Total attendance visits
* Last 5 attendance entries
* Attendance percentage chart

#### **3. Announcements & Events**

* Branch announcements
* District/Area/Mission announcements
* Events happening this week
* Important notices

#### **4. Sermons & Spiritual Resources**

* Latest sermons uploaded
* Search be topic/pastor/date
* Audio/video integration

#### **5. Personal Profile**

* Show basic details
* Quick link to profile editing
* Membership status

---

# ### **18.3 Member Profile Management**

Members can edit only **safe, non-administrative fields**.

### **Editable Fields**

* Phone number
* Email
* Profile picture
* Address
* Emergency contact info
* Profession
* Marital status
* Date of birth

### **Non-editable (Admin/Executive control only)**

* Member ID
* Church hierarchy (branch/district/area)
* Baptism records
* Spiritual history
* Roles held (executives must modify)
* Financial records
* Contribution history
* Pastoral notes
* Staff/pastor status

### **Password / PIN Management**

* Members log in using **Member ID + PIN**
* They can change their PIN from profile settings
* Mission/branch admins can reset PIN to default (`12345`)

---

# ### **18.4 Member Contribution Module**

Members cannot enter contributions manually — **branch admins handle that**.

Members can **only view**:

### **What Members Can See**

* All their contributions (individual + general)
* Status of pledges (Paid / Partial / Outstanding)
* Funeral contributions they participated in
* Yearly giving reports
* Printable/downloadable statements

### **What They Cannot Do**

* Add contributions
* Edit or delete contributions
* View other members’ contributions
* View branch balances or expenditure reports

---

# ### **18.5 Group & Department Assignment**

Members can view the ministries/committees they belong to:

### **Examples of Groups**

* Youth
* Women’s Fellowship
* Men’s Fellowship
* Choir
* Children Ministry
* Welfare/Visitation Teams

### **Members Can:**

* View groups they belong to
* See group announcements
* See group meeting dates (future feature)

### **Members Cannot:**

* Join/leave groups by themselves
* Create groups
* Manage group leadership

Admins control group assignment.

---

# ### **18.6 Announcements System (Member View)**

Members see all announcements relevant to them based on hierarchy:

### **Visible to Members**

* Branch announcements
* District announcements
* Area announcements
* Mission announcements

### **Features Available**

* Open announcement
* View attachments (PDF, images)
* Share to WhatsApp (optional)
* Saved/bookmarked announcements

Members **cannot create announcements**.

---

# ### **18.7 Sermon Library**

Members can access all sermons published by:

* Branch Pastor
* District sermons
* Area sermons
* Mission-wide national sermons

Includes:

* Text sermons
* Audio uploads
* YouTube/Video sermons
* Downloadable notes

---

# ### **18.8 Attendance Tracking (Member View)**

Members can see only **their own attendance**.

### **They Can View:**

* All attendance history
* Year-to-date attendance count
* Attendance streak (optional feature)
* Service type breakdown:

  * Sunday
  * Midweek
  * Youth/Men/Women
  * Special Programs

### **They Cannot:**

* Modify attendance
* Submit attendance
* View attendance of other members

---

# ### **18.9 Member Restrictions**

Members **cannot**, under any condition:

* Access financial modules
* View branch-level money or expenditure
* Access payroll
* Enter contributions
* Edit financial entries
* View audit logs
* Create announcements
* Manage hierarchy (branch/district/area)
* Access administrative dashboards

Strict security rules apply at the member level.

---

# ### **18.10 Member Registration Enhancements**

Members benefit from data captured during registration:

### **Optional Data (configured by Mission Admin)**

* Profile picture capture
* Emergency contact
* Profession
* Marital status
* Skills & talents (future module)

The cleaner the data, the better the analytics.

---

# ### **18.11 Member Year-End View**

At year-end rollover:

* Their entire contributions are moved to “Previous Year”
* They can still view past yearly statements
* Attendance for past years remains available

This keeps the profile clean and optimized.

---
