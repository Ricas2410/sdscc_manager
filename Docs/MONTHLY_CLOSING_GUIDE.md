# Monthly Closing Guide

## Overview
The Monthly Closing feature allows branch administrators to close their financial month, locking all contributions and expenditures for that period. This ensures financial integrity and prevents unauthorized changes.

## Access
- **URL:** `/monthly-closing/`
- **Permissions:** Branch Admin, Mission Admin
- **Menu:** Finance â†’ Monthly Closing

## Features

### 1. Month Selection
- Select branch (Mission Admin only)
- Select month and year
- View real-time financial summary

### 2. Financial Summary
- **Total Contributions:** All contributions for the month
- **Mission Allocation:** Amount due to mission headquarters
- **Expenditures:** Total expenses for the month
- **Net Balance:** Remaining funds at branch level

### 3. Close Month
- Click "Close Month" button
- System validates all data
- Locks all contributions/expenditures
- Calculates remittances
- Generates automatic notifications

### 4. After Closing
- Month cannot be edited (except by Mission Admin)
- Monthly report can be generated
- PDF export available
- Remittance status tracked

### 5. Reopen Month (Mission Admin Only)
- Mission Admin can reopen closed months
- Use for corrections or adjustments
- Audit trail maintained

## Edit Restrictions
- **24-hour window:** Edit contributions within 24 hours of creation
- **Closed month:** No edits after month is closed
- **Admin override:** Mission Admin can edit any time

## Reports
- **Monthly Report:** Comprehensive financial statement
- **PDF Export:** Print-ready format
- **Includes:** All contribution types, expenditures, totals

## Best Practices
1. Review all entries before closing
2. Ensure all expenditures are entered
3. Verify remittance calculations
4. Generate PDF report for records
5. Keep audit trail of closures

## Troubleshooting
- **Cannot close month:** Check for unapproved expenditures
- **Missing data:** Ensure all contributions have proper allocation
- **Permission denied:** Verify user role and permissions

## Support
Contact system administrator for assistance with month closing issues.
